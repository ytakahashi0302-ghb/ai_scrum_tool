## 2. 主要な変更点

### 2.1 レイアウト変更
- 旧: 水平タブバー (5タブ) による切替
- 新: 左サイドバー型マスターディテール UI
- モーダル幅: `5xl`
- サイドバー幅: 220–240px

### 2.2 情報アーキテクチャ再編

| カテゴリ | セクション |
|---------|-----------|
| 一般 | プロジェクト |
| AI & モデル | POアシスタント / AIプロバイダー / チーム設定 |
| システム | セットアップ状況 / アナリティクス |

### 2.3 コンポーネント分割
`GlobalSettingsModal.tsx` (1536行) を以下に分割：
- `SettingsShell` / `SettingsSidebar` / `SettingsSection` / `SettingsField` / `SettingsContext`
- `sections/ProjectSection` / `sections/PoAssistantSection` / `sections/AiProviderSection`
